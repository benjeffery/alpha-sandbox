expcat.plugins.QueryComposerUI(nestingDivs)
Builds the user interface DOM for composing CQL queries. This implementation allows only one level of nesting. The composer provides a method to build a DOM tree that can be attached to the document. The constructor creates a new composer.
Author: Jacob Almagro - ExplorerCat Project..
Parameters:
nestingDivs
{[jQuery]} An array containing the nesting divs (sorted from outer to inner) that will be cloned when building the DOM tree for the user interface.

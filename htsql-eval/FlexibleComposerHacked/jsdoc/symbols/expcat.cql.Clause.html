Class Detail
expcat.cql.Clause()
Represents a CQL expression (operator and operands) that can be evaluated as true/false. As you would expect clauses can be used to build other clauses. The constructor creates an empty clause that has to be configured using the setter methods.
Author: Jacob Almagro - ExplorerCat project..

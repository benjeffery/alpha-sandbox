expcat.plugins.OperatorBar(id, operatorSymbolMap, updateFunction)
An operator bar represents operator and operator that can be configured by the user via UI. The operator represented here joins two clauses or group of clauses (or conditions).
Author: Jacob Almagro - ExplorerCat Project.
Parameters:
id
{Integer} An integer that identifies uniquely the operator bar.
operatorSymbolMap
{Object} A hash that associates operator symbols to operators.
updateFunction
{Function} The callback that will be executed when the operator changes.

expcat.cql.Property()
Represents the definition of an entity property. They are basically named values that can have some additional attributes like maximum, minimum or a list of allowed values. The constructor builds a property from a property definition (object).
Author: Jacob Almagro - ExplorerCat Project..
Parameters:
definition.name
{String} The name of the property.
definition.type
{String} CQL type for the property value.
definition.description
{String} Brief description of the property.
definition.maximum
{Number} Maximum value it can take (undefined if there is no maximum).
definition.minimum
{Number} Minimum value it can take (undefined if there is no minimum).
definition.allowedValues
{Array} Array of permitted values for the property (undefined if there are no restrictions).

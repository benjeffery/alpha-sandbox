HTSQL Web Service

You can play with the HTSQL web service directly, e.g., try the following queries:

  • /snp.limit(10)
  • /snp.limit(10)/:json (JSON objects)
  • /snp.limit(10)/:raw (JSON arrays)
  • /snp.limit(10)/:csv
  • /snp.limit(10)/:tsv
  • /snp.limit(10)/:sql - see the actual SQL that the HTSQL query is rewritten to
  • /snp.filter(daf_sea>0.9&daf_afr<0.1).sort(daf_sea-)
  • /snp.filter(genetext~'pfcrt').sort(chromosome,position)
  • /snp.filter(chromosome='MAL1'&position>=1&position<=100000).sort(chromosome,position)
  • /snp^chromosome
  • /snp^chromosome{chromosome,count(snp)} or equivalent /snp^chromosome{*,count(^)}
  • /snp^chromosome{chromosome,count(snp.filter(mutation='SYN')) :as syn,count(snp.filter(mutation='NON')) :as non}
  • /snp^chromosome{chromosome,100*(count(snp.filter(daf_sea>0.8&daf_afr<0.2))/count(snp))}
  • /gene.limit(10)
  • /gene.filter(fulltext~'pfmdr').sort(chromosome,start)
  • /gene.filter(fulltext~'transport').sort(chromosome,start)
